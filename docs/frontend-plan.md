## 2. 路由规划

```
/login                         → 登录页
/                              → 仪表盘（今日概览）
/projects                      → 项目列表
/projects/:projectId           → 项目详情（船舶列表 + 成员）
/projects/:projectId/ships/:shipId  → 检验项目列表
/inspections/:id               → 检验详情（结果填写 + 意见管理）
/reports                       → 报表中心
/admin/users                   → 用户管理（admin）
/admin/projects                → 项目管理（admin/manager）
/admin/import-logs             → 导入日志与异常处理（admin）
```

## 4. 通用组件设计

| 组件 | 说明 | 复用位置 |
|------|------|---------|
| `ResultTag` | 检验结果彩色标签（CX/AA/QCC/OWC/RJ） | 列表、详情、报表 |
| `DisciplineTag` | 专业彩色标签 | 列表、详情、报表 |
| `StatusBadge` | 意见状态徽章（open/closed） | 意见列表 |
| `InspectionFilters` | 多维筛选器（专业/日期/结果/状态） | 检验列表、报表 |
| `CommentCard` | 意见卡片（内容 + 状态 + 操作按钮） | 检验详情 |
| `StatCard` | 统计数字卡片 | 仪表盘 |
| `PageHeader` | 页面标题 + 面包屑 | 所有页面 |
| `VersionConflictModal` | 乐观锁冲突提示弹窗 | 检验详情 |
